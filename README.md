# nmaphtmlreport
Python Program to do nmap scan and convert the output to html
Dependency:
1. Nmap
2. xsltproc

Flow of the code.
1. Run as sudo
2. Key in the target
3. Key in the filename
4. Open nmap_output.html in browser
